# Queue Pattern Recognition

## Common Queue Patterns

### 1. BFS
- Level Order Traversal
- Rotten Oranges
- Shortest Path (Unweighted Graph)

### 2. Sliding Window
- First Negative Integer
- Maximum of All Subarrays

### 3. Stream Processing
- First Non-Repeating Character
- Moving Average

### 4. Simulation
- Hot Potato Game
- Josephus Problem

### 5. Queue + Stack
- Reverse Queue
- Reverse First K Elements

### Recognition Clues

| Clue | Pattern |
|--------|---------|
| Level-wise processing | BFS |
| Fixed-size window | Sliding Window |
| Incoming stream | Queue |
| FIFO requirement | Queue |
| Nearest future event | Queue/Deque |
